# Генератор бомжей

Генерирует персонажей для Dark Heresy первой редакции.

Код — полная говнина, зато сразу готово к использованию.

Пока не окончен.

## Использование

 1. Скачать
 2. Открыть `index.html` в браузере
 3. Потыкать на все выборы
 4. Плакать
